This project is a beginner-friendly bioinformatics toolkit built using Python to perform essential DNA sequence analysis operations such as validation, transcription, and reverse complement generation.
# DNA-toolkit-python
A simple DNA Toolkit built in Python including validation, nucleotide frequency, transcription, and reverse complement functions

## Features
- DNA sequence validation
- Random DNA sequence generation
- Nucleotide frequency calculation
- DNA → RNA transcription
- Reverse complement generation

## Technologies Used
- Python
- VS Code
